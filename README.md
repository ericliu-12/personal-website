# Eric Liu — Terminal Portfolio

An interactive terminal-themed personal website. Type commands to explore.

**Live:** [ericliu-12.github.io/personal-website](https://ericliu-12.github.io/personal-website/)

## Commands

`help` · `about` · `experience` · `projects` · `skills` · `education` · `resume` · `contact` · `theme` · `neofetch` · `clear` · `history`

...and a few hidden easter eggs.

## Tech

Vanilla HTML + CSS + JS — no dependencies, no build step. Deployed via GitHub Pages.
